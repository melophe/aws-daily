Amazon AppFlow とは

Amazon AppFlow は、SaaS アプリケーション（Salesforce、Slack、ServiceNow、Zendesk など）や AWS サービス（S3、Redshift など）の間で、コードを書かずに安全にデータを双方向に転送・統合できるマネージドサービス です。

主な特徴

コードレスでデータ連携

GUI 上で数クリックするだけで、SaaS ⇔ AWS 間のデータフローを作成可能。

例：Salesforce から顧客データを取得し、S3 に保存。

双方向データ転送

SaaS → AWS

AWS → SaaS

双方向に対応。

データ変換やフィルタリング

フィールドマッピングや変換、フィルタを組み込める。

例：日付形式を統一、不要フィールドを削除、条件付き抽出。

セキュアな転送

転送データは自動的に暗号化。

SaaS との接続は AWS PrivateLink を利用可能（インターネットを通さずプライベート接続）。

スケジュール実行やイベント駆動

定期的な同期（毎時、毎日など）や、イベントが起きたときに自動的にデータ連携。

利用シーンの例

マーケティング分析
Salesforce の顧客データを S3 に自動連携し、Athena / QuickSight で可視化。

サポート運用
Zendesk のチケットデータを Redshift に流し込み、BI 分析に利用。

双方向連携
S3 にある製品マスタを ServiceNow に反映。

まとめ

AppFlow = SaaS ⇔ AWS のデータ連携サービス

データの収集・転送を「コードなしで」「セキュアに」「スケーラブルに」行える。
